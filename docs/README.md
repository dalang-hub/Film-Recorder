# docs文件夹

该文件夹包含项目文档和辅助文件

## 子文件夹说明

- **notes/**: 开发笔记和文档记录
  - FIX_NOTES.md: 修复记录
  - UI_UPDATE_NOTES.md: UI更新记录
  - UI_UPDATE_SUMMARY.md: UI更新总结
  - SHAPEAPPERANCE_FIX.md: ShapeAppearance修复记录
  - 项目最终状态.md: 项目状态总结

- **icons/**: 项目图标文件
  - app_icon.svg: 主应用图标
  - app_icon_clean.svg: 简洁版应用图标
  - app_icon_simplified.svg: 简化版应用图标

- **scripts/**: 辅助脚本
  - upload_to_github.bat: GitHub上传脚本

- **project_status/**: 项目状态文档
  - UI_应用记录.md: UI应用记录

## 使用说明

这些文件主要用于项目开发和维护记录，不属于应用源代码的一部分。
